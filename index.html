URL博客

《浅析 URL》 URL 包括：协议+域名或 IP+端口号+路径+查询字符串+锚点 协议：在浏览器与服务器间传送文档 域名：替代 IP 地址 端口：同一 IP 下提供不同的服务 路径：同一个服务器可以使用不同的路径得到不同的页面 查询字符串：同一个页面，不同的内容 锚点：同一个页面不同的部分，更加的细化 DNS 的作用：使域名和 IP 相互对应起来 IP 的作用：可以定位一台设备的地址 ping 命令: ping baidu.com 就会得到百度的 IP 域名： IP 的字符串 com 是顶级域名 ……com 是一级域名（俗称一级域名） www.....com 是三级域名（俗称二级域名）
